# KPGU Logo Instructions

## Logo File Location
Please place your KPGU logo image file in the `public` folder with the filename: **`kpgu-logo.png`**

## Logo Specifications
- **File Name**: `kpgu-logo.png`
- **Location**: `/public/kpgu-logo.png`
- **Recommended Size**: 200px height (width will scale proportionally)
- **Format**: PNG with transparency (preferred) or JPG
- **Aspect Ratio**: Any (will maintain aspect ratio)

## How to Add Your Logo

1. Obtain the KPGU logo file (PNG format recommended)
2. Place it in the `public` folder as `kpgu-logo.png`
3. The logo will automatically appear on all pages in the left corner of the header

## Fallback
If the logo file is not found, a text-based logo "KPGU" with colored letters will be displayed instead.

## Supported Pages
The logo appears on:
- Landing Page
- Login Page
- Signup Page
- Track Bus Page
- Student Dashboard
- Parent Dashboard
- Driver Dashboard
- Staff Dashboard
- Admin Dashboard

All pages now use the reusable Header component which includes the logo.
